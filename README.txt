# Running the DIS Drink Project ###

Assumes a working Python 3 installation (with python=python3 and pip=pip3).

(1) Run the code below to install the dependencies.
>$ pip install -r requirements.txt

(2) Import csv files to database 


(3) In the app.py-file, set your own database username and password

(4) Run Web-App
> flask run 

----------------------------------------------------------------------------------------------

# How to use the application

(1) When running the app you enter the front page, where it is possible to see the beers and its details 

(2) It is possible to add a beer to the list, in case your favorite beer is missing 

(3) In the beer table there is a button for view details. when pressing this you get directed to a beer detail side

(4) In the beer detail side it is possible to see who rated the beer and what it was rated 

(5) You can add your own rating of the beer, by entering your username, beerid and the rating
